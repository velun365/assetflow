# AssetFlow

## 프로젝트 소개
AssetFlow는 기업/기관 에서 도서, 문서, 장비 등의 자산을 등록하고, 직원들에게 대여, 예약, 반납, 연체, 이용자 요청을 관리하는 백오피스 시스템입니다.

단순 물품관리 서비스가 아니라, 사내 기업자산의 상태와 대여 흐름을 관리하는 업무 시스템을 목표로 작업하였습니다.

## 개발 목표
- 기업자산 등록 및 관리
- 개별 자산 품목 관리
- 대여 신청 및 반납 처리
- 사용자별 대여 이력 조회
- 관리자용 대여 현황 관리
- 이후 예약, 연체, 요청 티켓, 통계 기능 확장

## 기술 스텍
- Spring Boot 3.5.15
- Spring Data JPA
- MySQL
- Gradle
- Lombok

## 1차 MVP 범위
- User
- Asset
- AssetItem
- CateGory
- Loan
